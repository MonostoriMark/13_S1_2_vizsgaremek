2. Bevezetés

2.1 A feladat címe

StaySharp - Szálloda Foglalási Platform

2.2 A feladat rövid ismertetése

A projekt célja egy modern, biztonságos online szálloda-foglaló platform létrehozása, amely összeköti a vendégeket és a szállodákat. A rendszer lehetővé teszi a vendégek számára, hogy könnyedén keressenek és foglaljanak szobát dátum, város és létszám alapján. A szállodák számára átfogó admin felületet biztosít a foglalások, szobák, árak és szolgáltatások kezelésére, valamint számlagenerálásra. A platform integrál különböző online fizetési megoldásokat és egy innovatív, RFID alapú check-in/check-out rendszert.

3. Elvárások a feladattal kapcsolatban

3.1 Operációs rendszer, környezet

  • Fejlesztői környezet: Windows 10/11, Linux, macOS
  • Éles környezet: Linux szerver (Ubuntu 20.04 LTS vagy újabb)
  • Böngésző támogatás: Chrome, Firefox, Safari, Edge legfrissebb verziói

3.2 Felhasználandó technológiák

  • Backend: Django (Python) REST API
  • Frontend: Vue.js 3 (Composition API, Pinia state management)
  • Adatbázis: PostgreSQL
  • Hitelesítés: JWT (JSON Web Tokens)
  • Fizetési integráció: Stripe / Barion / SimplePay
  • RFID Hardver: Raspberry Pi Pico W + RFID RC522 modul
  • Verziókövetés: Git, GitHub
  • Üzemeltetés: Docker, Docker Compose

3.3 Megoldás formátuma

  • Teljes Git repository a forráskóddal
  • Docker compose konfiguráció
  • Részletes technikai dokumentáció (.md és .pdf)
  • Forráskód dokumentáció
  • Erőforrás-terv és munkaidő-nyilvántartás
  • Felhasználói dokumentáció és képernyőképek

3.4 Szoftverfejlesztés

A feladat egy háromrétegű webalkalmazás megtervezése és megvalósítása, amely magában foglalja a felhasználói (vendég) felületet, a szálloda admin felületet és a rendszer admin felületet. A fejlesztés során be kell tartani a modern web fejlesztési elveket (reszponzív design, RESTful API-k, biztonságos kódolás).

3.5 Modulok

  • Backend Modul (Django)
  • Frontend Modul (Vue.js)
  • RFID Hardver Modul (Prototípus)
  • Projektmenedzsment és Dokumentáció

4. Szoftver specifikáció

4.1 Megjelenés

  • Reszponzív design minden eszközön (mobil, tablet, desktop)
  • Modern, felhasználóbarát felület a vendégek számára
  • Áttekinthető dashboard a szálloda adminok számára
  • Egyszerű és hatékony felület a rendszer adminok számára

4.2 Funkciók

  1. Nyilvános Oldalak & Vendég Funkciók:
    • Regisztráció és bejelentkezés (JWT)
    • Szálloda és szoba kereső
    • Szoba részletes megtekintése
    • Foglalási folyamat
    • Online fizetés integráció
    • Felhasználói profil
  2. Szálloda Admin Felület:
    • Dashboard
    • Szobakezelés
    • Foglaláskezelés
    • Számlázás
    • Beállítások
  3. Rendszer Admin Felület:
    • Felhasználók és szállodák listázása
    • Új szálloda regisztrációk validálása
    • Számlázási adatok megtekintése
  4. RFID Check-in/check-out Rendszer (Prototípus):
    • Raspberry Pi Pico W + RFID olvasó
    • WebSocket kommunikáció a backenddel
    • Check-in és check-out funkciók
  5. Biztonsági Funkciók:
    • JWT token alapú hitelesítés
    • Bizalmas adatok titkosított tárolása
    • API végpontok védelme
    • SQL injection és XSS védelem
    • Regisztráció validáció
